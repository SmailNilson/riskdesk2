## Proposed Fixes (Priority Order)

### P0 — Critical (fix immediately)

**1. Add error handling to CompletableFuture.runAsync() calls**
```java
// MentorSignalReviewService.java lines 196, 256, 327
// BEFORE:
CompletableFuture.runAsync(() -> analyzeAndPersist(saved.getId(), reviewPayload));

// AFTER:
CompletableFuture.runAsync(() -> analyzeAndPersist(saved.getId(), reviewPayload))
    .exceptionally(ex -> {
        log.error("Async analysis failed for review {}", saved.getId(), ex);
        return null;
    });
```

**3. Guard linkedMap() against odd argument count**
```java
private Map<String, Object> linkedMap(Object... values) {
    if (values.length % 2 != 0) {
        throw new IllegalArgumentException("linkedMap requires even number of arguments, got " + values.length);
    }
    Map<String, Object> map = new LinkedHashMap<>();
    for (int i = 0; i < values.length; i += 2) {
        map.put(String.valueOf(values[i]), values[i + 1]);
    }
    return map;
}
```
